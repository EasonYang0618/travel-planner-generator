# Travel Planner Generator

DTS114TC AI Software Engineering coursework project.

This project builds an AI-powered system for meta-software development. It generates SDLC documentation, UML diagrams, a Flask API, a travel planning website, and deployment assets.

To run the notebook, use Python/Jupyter with `requests`, `python-dotenv`, `flask`, `flask-cors`, and `pytest`. The APIFree key should be provided as `APIFREE_API_KEY`.
