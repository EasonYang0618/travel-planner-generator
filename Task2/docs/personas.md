1) Role: Solo weekend explorer (young professional)
   - Goal: Quickly generate a 2–3 day itinerary that maximises highlights and fits a mid-range budget and active interests (food, nightlife).
   - Main concern: Fast, clear results with logistic details (timings, travel between spots) so planning takes under 30 minutes.

2) Role: Budget backpacker
   - Goal: Get a low-cost, flexible multi-day plan focused on free/cheap attractions, public transport and local experiences.
   - Main concern: Accurate cost estimates and easy options for swapping activities if something is closed or weather changes.

3) Role: Family planner (parent travelling with kids)
   - Goal: Build a child-friendly, paced itinerary with suitable meal breaks, indoor options and safety considerations for a specified trip length.
   - Main concern: Safety, accessibility and realistic daily schedules (not too rushed), plus clear notes on child-appropriate activities.

4) Role: Travel agency staff / product user (admin)
   - Goal: Use the web UI or Flask API to generate customised itineraries for clients, possibly batch-create plans and export/print them.
   - Main concern: Reliable, stable API responses, predictable customization options (budget/age preferences), and easy editing/exporting for client presentation.
